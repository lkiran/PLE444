Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages

 
<alert type="{{alert.type}}"><span ng-bind-html="alert.msg | safe_html"></alert>

// in the controller
$scope.alert = { type: 'success', msg: 'Well done! You successfully read this important alert message.' };

Alerts can contain more than a line of messege. They can be extended with to focus on a particular messege.
