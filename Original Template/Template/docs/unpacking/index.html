Every downloadable zip contains the following folder structures:

Avenxo
|- admin_html/      Admin Template
|- frontend_html/   Frontend Template
|- landing_html/    Landing Page Template
|- documentation/   Documentation
|- psd/             Photoshop Files
|- readme.txt

Please also save the generated license and license-code. For item support, we may require you to provide the license-code.

Among all the themes, the basic folder structure remains the same

assets/
|- css/         The compiled LESS files into CSS
|- demo/        Images and Demo JS used within the theme. Can be removed.
|- fonts/       Font Icons used in Avenxo
|- img/         Images used at parts of the template, usually not required
|- js/          Includes the core JS files necessary for Avenxo
|- less/        Includes Bootstrap and Avenxo styles
|- plugins/     Includes all third-party plugins used in the them
